 随机首包密钥，随机生成seed，使数据更加安全

### 已实现的功能：
- 走路模拟器
- 队伍管理
- 传送
- 抽卡模拟器
- 背包
- 初级战斗
- 养成
- 初级模拟宇宙（1.6.0不支持）
- 副本/周本/挑战
- 忘却之庭

### 已实现的活动：
- 角色试用
- 巡星之礼
- 忘却之庭-虚构叙事

### 使用方法：
1. 拉取项目文件
2. run
```
  go mod tidy 
  go build main.go
  main / main.exe
```
