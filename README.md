 随机首包密钥，随机生成seed，使数据更加安全

### 已实现的功能：
- 走路模拟器
- 队伍管理
- 传送
- 抽卡模拟器
- 背包
- 初级战斗
- 养成
- 初级模拟宇宙(进入)
- 副本/周本/挑战
- 忘却之庭

### 已实现的活动：

2. run
```
  run build.bat / build.sh

```
